LRMError

NSError support for LeoRM.


LeoRM reports public failures through NSError. This header defines the LeoRM error domain, common error codes, userInfo keys, and helper functions used to build Cocoa-shaped errors from SQLite failures.
SQLite errors should preserve enough context to be useful without stepping into LeoRM internals: SQLite result code, SQLite message, SQL string where available, and database path where available.

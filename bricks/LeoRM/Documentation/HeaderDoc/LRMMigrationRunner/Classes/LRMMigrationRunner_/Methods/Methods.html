migrateDatabase:schema:error:

Migrates an open database to a schema's target version. 

- (BOOL)migrateDatabase:(LRMDatabase *)database schema:(LRMSchema *)schema error:(NSError **)error;
Parameters
database
Open LRMDatabase. Must not be nil and must be open.
schema
Schema description. Must not be nil.
error
Optional NSError output. Filled on invalid input, missing migration step, invalid version state, transaction failure, migration failure, or metadata update failure.
Return Value

Returns YES when the database is at the target version, NO on failure.

Discussion

If the current version already equals the target version, this method returns YES without applying migrations. If the current version is newer than the target version, the method returns NO.

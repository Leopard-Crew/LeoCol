LRMResultSet

Forward-only result set for one prepared statement.

Superclass: NSObject
Declared In: LRMResultSet


LRMResultSet owns query iteration for an LRMStatement. It retains the statement while open, steps through rows, exposes the current row, and finalizes the statement when closed.
Result sets are forward-only. They do not cache all rows. Callers should consume rows before advancing to the next row or closing the result set.
Instances use manual retain/release. Call close explicitly when finished. dealloc performs defensive cleanup.

LRMSchema

Named schema description for migration execution.

Superclass: NSObject
Declared In: LRMSchema


LRMSchema groups a schema name, a target version, and an ordered set of LRMMigration objects. It does not create tables by itself. It is input for LRMMigrationRunner.
A schema name identifies the version stored in LeoRM metadata. Domain stores should choose stable schema names.
Instances are immutable after initialization and use manual retain/release.

initWithName:targetVersion:migrations:error:

Initializes a schema description. 

- (id)initWithName:(NSString *)name targetVersion:(NSInteger)targetVersion migrations:(NSArray *)migrations error:(NSError **)error;
Parameters
name
Schema name. Must not be nil or empty.
targetVersion
Target schema version. Must not be negative.
migrations
Array containing only LRMMigration objects. The array is copied.
error
Optional NSError output. Filled on invalid input.
Return Value

Returns an owned LRMSchema, or nil on failure.

migrations

Returns the migration list. 

- (NSArray *)migrations;
Return Value

Returns the copied migration array. The caller does not own the returned object.

name

Returns the schema name. 

- (NSString *)name;
Return Value

Returns the copied schema name. The caller does not own the returned object.

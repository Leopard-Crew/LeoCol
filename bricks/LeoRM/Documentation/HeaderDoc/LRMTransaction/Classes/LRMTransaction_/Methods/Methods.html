commit:

Commits the active transaction. 

- (BOOL)commit:(NSError **)error;
Parameters
error
Optional NSError output. Filled when the transaction is inactive or COMMIT fails.
Return Value

Returns YES on success, NO on failure.

Discussion

After a successful commit the transaction becomes inactive. Calling commit: or rollback: again is an API error and returns NO.

initWithDatabase:error:

Begins a transaction on an open database. 

- (id)initWithDatabase:(LRMDatabase *)database error:(NSError **)error;
Parameters
database
Open LRMDatabase. Must not be nil and must be open.
error
Optional NSError output. Filled when the database is invalid or BEGIN TRANSACTION fails.
Return Value

Returns an owned active LRMTransaction, or nil on failure.

Discussion

This initializer executes BEGIN TRANSACTION. The transaction is active only after that SQL succeeds.

isActive

Reports whether the transaction is still active. 

- (BOOL)isActive;
Return Value

Returns YES between successful begin and successful commit or rollback.

rollback:

Rolls back the active transaction. 

- (BOOL)rollback:(NSError **)error;
Parameters
error
Optional NSError output. Filled when the transaction is inactive or ROLLBACK fails.
Return Value

Returns YES on success, NO on failure.

Discussion

After a successful rollback the transaction becomes inactive. Calling commit: or rollback: again is an API error and returns NO.

bindArguments:toStatement:error:

Binds an NSArray of positional arguments to a statement. 

- (BOOL)bindArguments:(NSArray *)arguments toStatement:(LRMStatement *)statement error:(NSError **)error;
Parameters
arguments
Optional NSArray of values. Passing nil is valid and binds no values.
statement
Statement to bind. Must not be nil.
error
Optional NSError output.
Return Value

Returns YES when all arguments bind successfully, NO on invalid state or binding failure.

Discussion

This helper binds values using LRMStatement bindObject:atIndex:error:. The first array object is bound at SQLite index 1.

database

Returns the repository database. 

- (LRMDatabase *)database;
Return Value

Returns the retained database. The caller does not own the returned object.

executeSQL:arguments:error:

Executes non-row-returning SQL with positional arguments. 

- (BOOL)executeSQL:(NSString *)sql arguments:(NSArray *)arguments error:(NSError **)error;
Parameters
sql
SQL string to execute. Must not be nil or empty.
arguments
Optional NSArray of values to bind. Values are bound in order starting at SQLite parameter index 1. Passing nil is valid and means no arguments.
error
Optional NSError output.
Return Value

Returns YES on success, NO on prepare, bind, or execution failure.

Discussion

This method keeps SQL visible. It is intended for repository methods such as insert, update, delete, and schema creation helpers.

initWithDatabase:error:

Initializes a repository with an open database. 

- (id)initWithDatabase:(LRMDatabase *)database error:(NSError **)error;
Parameters
database
Open LRMDatabase. Must not be nil and must be open.
error
Optional NSError output. Filled when the database is invalid.
Return Value

Returns an owned LRMRepository, or nil on failure.

Discussion

The repository retains the database. It does not open or close the database.

resultSetForSQL:arguments:error:

Prepares row-returning SQL and returns a result set. 

- (LRMResultSet *)resultSetForSQL:(NSString *)sql arguments:(NSArray *)arguments error:(NSError **)error;
Parameters
sql
SQL query string. Must not be nil or empty.
arguments
Optional NSArray of values to bind. Passing nil is valid.
error
Optional NSError output.
Return Value

Returns an autoreleased LRMResultSet, or nil on prepare, bind, or query setup failure.

Discussion

The caller should close the returned result set when finished. Closing the result set finalizes the underlying statement.

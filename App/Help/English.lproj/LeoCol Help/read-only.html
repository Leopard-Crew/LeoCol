Read-only Boundary

LeoCol observes and reports. It does not repair or modify the system.

LeoCol does not

  • kill processes,
  • delete files,
  • clean the system,
  • repair printers,
  • unload kernel extensions,
  • modify LaunchAgents or LaunchDaemons,
  • modify Login Items,
  • install background services,
  • run as a daemon.

This boundary is intentional. LeoCol is designed to make observations understandable without taking destructive action.

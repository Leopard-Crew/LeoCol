Snapshots and Process Instances

Use Update Snapshot to create a new process observation. LeoCol records what it sees, updates lifecycle information, and refreshes the viewer.

The Snapshot Overview lists recorded snapshots and their process counts.

Why processes can appear more than once

Some services run several instances at the same time. Other applications restart and receive a new PID. LeoCol keeps these technical instances visible instead of merging them automatically.

Instance status

  • Current: seen in the newest snapshot.
  • Historical: absent from a later snapshot.

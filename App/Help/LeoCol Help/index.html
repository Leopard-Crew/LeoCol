LeoCol Help

LeoCol is a read-only system observation tool for Mac OS X Leopard.

It records explicit snapshots and presents process lifecycle information, provenance evidence, snapshot history, and plain text reports.

LeoCol does not clean, repair, delete, kill, unload, install, or run as a daemon.

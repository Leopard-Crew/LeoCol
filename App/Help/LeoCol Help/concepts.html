Core Concepts

Snapshot

A snapshot is an explicit observation at one point in time. LeoCol is not a continuous live monitor.

Process instance

A process instance is identified by a PID and lifecycle information. Restarted applications or services may appear as separate historical and current instances.

Current

Current means the process instance was seen in the newest LeoCol process snapshot.

Historical

Historical means the process instance was seen before, but was absent from a later snapshot.

Last seen

Last seen means the last snapshot in which this PID was actually observed. It does not necessarily mean the process is running at this exact moment.

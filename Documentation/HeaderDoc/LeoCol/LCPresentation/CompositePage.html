LCPresentation

Canonical-value presentation helpers for LeoCol.

Discussion

LCPresentation maps stored technical values to localized user-facing strings without changing the underlying database values.

LCPresentationStringForValue

Maps canonical LeoCol values to localized presentation strings. 

NSString *LCPresentationStringForValue(
    id value,
    NSString *key,
    BOOL detail);
Parameters
value
The canonical value to present.
key
The row key that describes the value category.
detail
Pass YES for detail/inspector presentation, NO for table cells.
Return Value

A localized presentation string.

Discussion

The database keeps canonical technical values such as "resolved", "observed-only", "present", or "receipt-bom". This helper translates those values into user-facing strings for the Cocoa viewer without changing the stored data.

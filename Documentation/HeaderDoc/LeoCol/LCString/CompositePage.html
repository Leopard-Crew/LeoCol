LCString

Localization helper for LeoCol.

Discussion

LCString resolves Localizable.strings keys for the Cocoa application layer.


Functions

LCString

LCString

Returns a localized string for a LeoCol localization key. 

NSString *LCString(
    NSString *key);
Parameters
key
The localization key to resolve.
Return Value

The localized string, or the key itself when no localized value exists.

Discussion

LCString is the small application-level localization helper used by the Cocoa viewer. It resolves keys through Localizable.strings and deliberately keeps localization access out of the view/controller code.

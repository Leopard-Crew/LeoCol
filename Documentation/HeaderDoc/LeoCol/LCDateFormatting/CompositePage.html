LCDateFormatting

Date and timestamp presentation helpers for LeoCol.

Discussion

These functions convert LeoCol's canonical database timestamps into localized display strings for tables, inspectors, and reports.


Functions

LCDisplayCompactTimestampString
Formats a canonical LeoCol timestamp for compact table display.
LCDisplayTimestampString
Formats a canonical LeoCol timestamp for inspector display.

LCDisplayCompactTimestampString

Formats a canonical LeoCol timestamp for compact table display. 

NSString *LCDisplayCompactTimestampString(
    NSString *timestamp);
Parameters
timestamp
A canonical timestamp string.
Return Value

A compact localized date/time string, or an empty string when no value is available.

Discussion

The input timestamp is expected to use LeoCol's canonical database format. The returned value follows the user's current system locale.

LCDisplayTimestampString

Formats a canonical LeoCol timestamp for inspector display. 

NSString *LCDisplayTimestampString(
    NSString *timestamp);
Parameters
timestamp
A canonical timestamp string.
Return Value

A localized date/time string, or "-" when no value is available.

Discussion

This formatter is used for the Process Details inspector where a slightly more descriptive localized date/time representation is appropriate.

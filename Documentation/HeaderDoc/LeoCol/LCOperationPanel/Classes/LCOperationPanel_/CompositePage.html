LCOperationPanel

Small reusable progress panel for explicit LeoCol operations.

Superclass: NSObject
Declared In: LCOperationPanel

Discussion

LCOperationPanel is used for user-triggered operations such as Update Snapshot and Update Evidence. It provides visible progress feedback so LeoCol does not appear frozen while helper tools run.


Methods

-appendLogLine:
Appends one line to the read-only operation log.
-beginOperation
Puts the panel into running state and starts the progress indicator.
-completeOperationWithStatus:
Stops progress indication and enables the Done button.
-setStatusText:
Updates the visible operation status line.
-showWithTitle:status:
Shows the operation panel with an initial title and status.

LCStoreSupport

Shared store support helpers for LeoCol's LeoRM-backed stores.

Superclass: NSObject
Declared In: LCStoreSupport

Discussion

LCStoreSupport centralizes project path, database path, and database open logic for the Cocoa application stores. It keeps LeoRM usage consistent without hiding SQL or domain-specific row mapping.

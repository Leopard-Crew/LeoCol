LCProvenanceStore

Loads provenance evidence summaries for the Cocoa viewer.

Superclass: NSObject
Declared In: LCProvenanceStore

Discussion

LCProvenanceStore reads grouped evidence counts from provenance_evidence. It does not run probes and does not modify the system.


Methods

Parameters
statusString
Optional output parameter receiving a user-facing status string.
Return Value

An array of evidence summary row dictionaries.

Discussion

Returned rows are grouped by evidence type and resolution state.

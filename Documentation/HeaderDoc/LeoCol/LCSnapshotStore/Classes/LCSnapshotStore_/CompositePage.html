LCSnapshotStore

Loads process snapshot summaries for the Cocoa viewer.

Superclass: NSObject
Declared In: LCSnapshotStore

Discussion

LCSnapshotStore reads snapshot_run and process_observation from the LeoCol database. It provides read-only summary rows for the planned Snapshot Overview panel.


Methods

+ (NSArray *)loadSnapshotSummaryRowsWithStatusString:(NSString **)statusString;
Parameters
statusString
Optional output parameter receiving a user-facing status string.
Return Value

An array of snapshot summary row dictionaries.

Discussion

Returned rows contain snapshot ID, observed timestamp, source, and process count. This makes LeoCol's snapshot-based model visible to the user.

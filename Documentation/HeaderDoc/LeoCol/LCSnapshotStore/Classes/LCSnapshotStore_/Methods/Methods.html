loadSnapshotSummaryRowsWithStatusString:

Loads snapshot summary rows from the LeoCol database. 

+ (NSArray *)loadSnapshotSummaryRowsWithStatusString:(NSString **)statusString;
Parameters
statusString
Optional output parameter receiving a user-facing status string.
Return Value

An array of snapshot summary row dictionaries.

Discussion

Returned rows contain snapshot ID, observed timestamp, source, and process count. This makes LeoCol's snapshot-based model visible to the user.
